# рҹҗӣ TROUBLESHOOTING

## Error: "Database connection failed"
**Solusi:**
1. Cek di Variables apakah `DB_HOST` dll sudah benar
2. Pastikan database sudah рҹҹў (bukan рҹҹЎ)
3. Redeploy aplikasi: Settings вҶ’ Deploy вҶ’ Redeploy

## Error: "500 Server Error"
**Solusi:**
Di Console, jalankan:
```bash
php artisan cache:clear
php artisan config:cache
php artisan view:cache
```

## Error: "APP_KEY not set"
**Solusi:**
1. Generate key di local: `php artisan key:generate --show`
2. Copy hasilnya ke Railway variable `APP_KEY`
3. Redeploy aplikasi

## Error: "Permission denied storage"
**Solusi:**
Di Console, jalankan:
```bash
chmod -R 775 storage/
chmod -R 775 bootstrap/cache/
```
